Detailed Description

template<class T>
class JackRingBuffer< T >

This is a C++ abstraction over jack_ringbuffer_t. It is a template class that allows reading and writing of structs of a given type T. T has to be a data type that can be copied just by copying the memory that its objects use (with size equal to sizeof(T)). E.g., classes with constructor and/or destructor generally don't qualify. Simple structs generally do.

Calling this method does not advance the read pointer, i.e., calling this multiple times successively always returns the same. Note: this method involves copying the data from the ring buffer to the given buffer, thus do not use it more often than necessary.
